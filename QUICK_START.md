# 🚀 БЫСТРЫЙ СТАРТ

## ⚡ За 3 шага запустите полную систему!

---

### ШАГ 1: Запустите Backend

Откройте терминал 1:

```bash
cd backend
npm run dev
```

✅ Должны увидеть:
```
✅ Сервер успешно запущен!

   URL: http://localhost:3001
   
🔑 Учетные записи:
   admin / admin123 (администратор)
   user / user123 (пользователь)
```

**НЕ ЗАКРЫВАЙТЕ этот терминал!**

---

### ШАГ 2: Запустите Frontend

Откройте терминал 2 (из корня проекта):

```bash
npm run dev
```

✅ Должны увидеть:
```
  ➜  Local:   http://localhost:5173/
```

**НЕ ЗАКРЫВАЙТЕ этот терминал!**

---

### ШАГ 3: Откройте браузер

Откройте: **http://localhost:5173**

1. Введите:
   - **Username:** `admin`
   - **Password:** `admin123`
2. Нажмите "Войти"

✅ **ГОТОВО!** Вы в системе! 🎉

---

## 🧪 ЧТО ПРОТЕСТИРОВАТЬ:

### 1. Авторизация ✅
   - Вход как admin
   - Выход
   - Вход как user

### 2. Управление пользователями ✅ (только admin)
   - Кликните "Пользователи" в меню
   - Создайте нового пользователя
   - Удалите пользователя

### 3. Темы ✅
   - Переключите светлую/темную тему
   - Тема сохраняется

---

## 🎯 УЧЕТНЫЕ ЗАПИСИ:

| Username | Password   | Роль  | Права |
|----------|------------|-------|-------|
| admin    | admin123   | Admin | Полный доступ + управление пользователями |
| user     | user123    | User  | Просмотр данных (нет доступа к управлению) |

---

## ❓ ПРОБЛЕМЫ?

### "Ошибка подключения к серверу"
→ Проверьте что Backend запущен на `http://localhost:3001`

### "Неверный логин или пароль"
→ Используйте `admin` / `admin123`

### CORS ошибка
→ Backend должен быть на порту 3001, Frontend на 5173
