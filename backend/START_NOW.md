## 🚀 ТРИ КОМАНДЫ ДЛЯ ЗАПУСКА:

### 1. Удалите старую установку:

```bash
cd backend
rmdir /s /q node_modules
```

**Если не удаляется:**
- Закройте VSCode
- Закройте все терминалы
- Попробуйте снова

### 2. Установите зависимости:

```bash
npm install
```

Должно пройти **БЕЗ ОШИБОК!** ✅

### 3. Настройте и запустите:

```bash
# Скопируйте пример .env
copy .env.example .env

# Откройте .env в редакторе и заполните ЭТИ 3 переменные:
# INFLUX_TOKEN=ваш_токен
# INFLUX_ORG=ваша_организация  
# JWT_SECRET=любая_длинная_случайная_строка_минимум_32_символа

# Инициализируйте базу данных
npm run migrate

# Запустите сервер
npm run dev
```

## 🐛 ЕСЛИ НЕ РАБОТАЕТ:

### npm install дает ошибки:

1. Проверьте что в `package.json` **НЕТ** `better-sqlite3`
2. Удалите `package-lock.json`
3. Запустите `npm install` снова

### EADDRINUSE (порт занят):

Откройте `.env` и измените:
```env
PORT=3002
```

### "Переменная окружения не установлена":

Откройте `.env` и заполните обязательные переменные:
- `INFLUX_TOKEN`
- `INFLUX_ORG`
- `JWT_SECRET`

### Другая ошибка:

Пришлите мне текст ошибки!
