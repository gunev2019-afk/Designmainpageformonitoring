# 🪟 ИНСТРУКЦИЯ ДЛЯ WINDOWS

## 🎯 Что случилось?

При установке `npm install` возникла ошибка компиляции пакета `better-sqlite3`.

**Причина:** На Windows этот пакет требует Visual Studio Build Tools с компонентом C++.

**Решение:** Мы переделали backend на **JSON базу данных** для быстрого тестирования!

---

## ✅ ЧТО ТЕПЕРЬ РАБОТАЕТ:

- ✅ **Никакой компиляции** - все работает сразу
- ✅ **JSON файл** вместо SQLite
- ✅ **Полностью функциональная БД** для разработки
- ✅ **Легко смотреть данные** - обычный JSON файл

---

## 🚀 ЗАПУСК (3 ШАГА):

### 1. Удалите старые файлы:

```bash
# В PowerShell или CMD
cd backend

# Удалите node_modules (если есть)
rmdir /s /q node_modules
```

Если не удаляется - закройте VSCode и все терминалы, попробуйте снова.

### 2. Установите зависимости:

```bash
npm install
```

Должно пройти **БЕЗ ОШИБОК!** ✅

### 3. Настройте и запустите:

```bash
# Создайте .env
copy .env.example .env

# Откройте .env и заполните:
# - INFLUX_TOKEN
# - INFLUX_ORG  
# - JWT_SECRET

# Инициализируйте БД
npm run migrate

# Запустите сервер
npm run dev
```

### 4. Проверьте:

Откройте: **http://localhost:3001**

## 🐛 Частые проблемы:

### 1. "Cannot find module"
```bash
npm install
```

### 2. "EADDRINUSE: порт занят"
Измените в `.env`:
```env
PORT=3002
```

### 3. Не могу удалить node_modules
1. Закройте VSCode
2. Закройте все терминалы
3. Перезагрузите компьютер
4. Попробуйте снова

### 4. npm install все еще дает ошибки
Убедитесь что в `package.json` **НЕТ** строки:
```json
"better-sqlite3": "..."
```

Если есть - я уже её удалил, но может быть старый файл.

## 🔄 Переход на SQLite (позже):

Когда будете готовы или развернете на Linux VPS:

### Вариант А: Установить Build Tools на Windows

1. Скачайте: https://visualstudio.microsoft.com/downloads/#build-tools-for-visual-studio-2022
2. Установите с компонентом "Desktop development with C++"
3. Добавьте в package.json: `"better-sqlite3": "^9.2.2"`
4. Запустите `npm install`

### Вариант Б: Развернуть на Linux

На Linux SQLite работает из коробки, без проблем!

```bash
# На Ubuntu/Debian
npm install
# Все установится без ошибок!
```
