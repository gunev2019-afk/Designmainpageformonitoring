# ⚡ БЫСТРОЕ ИСПРАВЛЕНИЕ

## 🔴 Проблема 1: Сразу попадаете на главную без авторизации

### ✅ Решение (3 секунды):

1. Откройте сайт http://localhost:5173
2. Нажмите **F12**
3. В консоли введите:

```javascript
localStorage.clear()
location.reload()
```

4. Готово! Теперь вы на странице входа

---

## 🔴 Проблема 2: На графиках ничего не отображается

### 🔍 Диагностика (открыть консоль):

1. Нажмите **F12**
2. Перейдите в **Console**
3. Обновите страницу
4. Найдите логи:

```
📥 Запрос данных: {...}
📈 Получены данные: {...}
✅ Данных получено: XX записей
```

---

### ⚠️ ВАРИАНТ А: Видите "✅ Данных получено: 0 записей"

**Причина:** В InfluxDB нет данных за выбранный период

**Решение:**

#### 1. Проверьте InfluxDB

Откройте http://localhost:8086 → **Data Explorer**

Выполните запрос:
```flux
from(bucket: "telemetry")
  |> range(start: -1h)
  |> filter(fn: (r) => r._measurement == "lab")
  |> filter(fn: (r) => r._field == "температура" or r._field == "влажность")
  |> limit(n: 10)
```

**Если пусто** → нужно добавить данные в InfluxDB!

**Если есть данные** → проверьте параметры станции:

#### 2. Проверьте параметры станции

В консоли браузера найдите:
```
📥 Запрос данных: {stationId: 1, ...}
```

Затем F12 → **Network** → найдите `/api/stations/1` → **Response**:

```json
{
  "data": {
    "bucket": "telemetry",
    "measurement": "lab"
  }
}
```

**Проверьте:** `bucket` и `measurement` должны совпадать с вашими данными в InfluxDB!

#### 3. Проверьте параметры метрик

Network → `/api/metrics` → Response:

```json
{
  "data": [
    {
      "field": "температура",
      "channel": "AI1"
    }
  ]
}
```

**Проверьте:** `field` и `channel` должны совпадать с тегами в InfluxDB!

### ⚠️ ВАРИАНТ В: Видите ошибку в консоли

```
❌ Ошибка загрузки данных: ...
```

**Решение:**

1. Проверьте что **Backend запущен**:
   ```bash
   cd backend
   npm run dev
   ```

2. Проверьте что **InfluxDB запущен**:
   - Откройте http://localhost:8086
   - Должна открыться страница InfluxDB

3. Проверьте `.env` файл:
   ```env
   INFLUX_URL=http://localhost:8086
   INFLUX_TOKEN=ваш_токен
   INFLUX_ORG=ваша_организация
   INFLUX_BUCKET=telemetry
   ```

## 🆘 ЭКСТРЕННАЯ ПОМОЩЬ:

### Если совсем ничего не работает:

1. **Остановите backend** (Ctrl+C)
2. **Очистите кэш:**
   ```bash
   cd backend
   rmdir /s /q node_modules\.cache
   ```
3. **Запустите снова:**
   ```bash
   npm run dev
   ```

4. **Очистите браузер:**
   - F12 → Console
   - `localStorage.clear()`
   - `location.reload()`

5. **Создайте ТЕСТОВУЮ станцию:**
   - Войдите как admin/admin123
   - Станции → Создать
   - Bucket: `telemetry`
   - Measurement: `lab`
   - Добавьте метрики с вашими field/channel из InfluxDB

---

📝 **Если проблема остается, пришлите:**
1. Скриншот консоли браузера (F12 → Console)
2. Скриншот Network → `/api/data/history` → Response
3. Скриншот логов backend терминала
