## 🐛 ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ:

### Проблема: "Ошибка подключения к серверу"

**Решение:**
1. Проверьте что Backend запущен:
   ```bash
   cd backend
   npm run dev
   ```
2. Проверьте что Backend на порту 3001:
   ```
   http://localhost:3001
   ```
3. Откройте консоль браузера (F12) и посмотрите ошибки

---

### Проблема: "Неверный логин или пароль"

**Решение:**
1. Проверьте что миграция выполнена:
   ```bash
   cd backend
   npm run migrate
   ```
2. Используйте правильные учетные данные:
   - **Admin:** `admin` / `admin123`
   - **User:** `user` / `user123`

---

### Проблема: CORS ошибка

**Решение:**
1. Проверьте `.env` в `backend/`:
   ```
   CORS_ORIGIN=http://localhost:5173
   ```
2. Перезапустите Backend сервер

---

### Проблема: Токен не сохраняется

**Решение:**
1. Откройте DevTools (F12)
2. Application → Local Storage
3. Проверьте что есть ключ `token`
4. Если нет - очистите localStorage и войдите заново
