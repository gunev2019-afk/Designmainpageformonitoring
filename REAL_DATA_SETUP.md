### **Шаг 4: Проверьте данные в InfluxDB**

Ваши данные в InfluxDB должны иметь такую структуру:

```flux
// Пример запроса:
from(bucket: "telemetry")
  |> range(start: -1h)
  |> filter(fn: (r) => r._measurement == "lab")
  |> filter(fn: (r) => r._field == "температура" or r._field == "влажность")
  |> filter(fn: (r) => r.channel == "AI1" or r.channel == "AI2" or r.channel == "AI3")
```

**Требуемая структура:**
- `_measurement`: `lab`
- `_field`: `температура` или `влажность`
- `channel` (tag): `AI1`, `AI2`, или `AI3`
- `_value`: числовое значение
