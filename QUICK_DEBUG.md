### 1. **Обновите страницу** (Ctrl+Shift+R)

### 2. **Откройте консоль** (F12 → Console)

### 3. **Найдите блок загрузки данных:**

```

```

---

## 🔍 ПРОВЕРКА ПО ШАГАМ:

### ✅ Шаг 1: Проверьте запрос

Найдите:
```
📥 ========== ЗАПРОС ИСТОРИЧЕСКИХ ДАННЫХ ==========
📥 Параметры: {
  stationId: 1,
  metricIds: [1, 2, 3],
  timeRange: "1h",
  frequency: "1m"
}
```

**❌ Если `metricIds: []`** → Метрики не выбраны!

---

### ✅ Шаг 2: Проверьте ответ

Найдите:
```
📈 ========== ОТВЕТ С ИСТОРИЧЕСКИМИ ДАННЫМИ ==========
📈 Полный ответ: {success: true, data: [...]}
✅ Данных получено: XX записей
```

#### **❌ Если `Данных получено: 0 записей`**

```
✅ Данных получено: 0 записей
```

**→ В InfluxDB НЕТ исторических данных за выбранный период!**

**Проверьте:**
1. Откройте InfluxDB → http://localhost:8086
2. Data Explorer
3. Выполните запрос:
```flux
from(bucket: "telemetry")
  |> range(start: -1h)
  |> filter(fn: (r) => r._measurement == "lab")
```

**Если данных нет** → Добавьте тестовые данные!

---

### ✅ Шаг 3: Проверьте структуру данных

Найдите:
```
🔍 Первая запись: {timestamp: "...", ...}
🔑 Найдены ключи метрик: ["metric_1", "metric_2", "metric_3"]
  - metric_1: 22.5
  - metric_2: 23.1
  - metric_3: 45.2
```

#### **❌ КРИТИЧЕСКАЯ ОШИБКА: Найдены ключи метрик: []**

```
🔍 Первая запись: {timestamp: "2026-03-17T12:30:00Z"}
🔑 Найдены ключи метрик: []
❌ КРИТИЧНО: В данных НЕТ полей metric_X!
```

**→ Backend НЕ ДОБАВЛЯЕТ значения метрик в ответ!**

**Это означает:**
- ✅ InfluxDB возвращает данные
- ❌ Backend НЕ преобразует их в формат `metric_1`, `metric_2`

**Решение:**
1. Проверьте backend/src/routes/data.ts
2. Функция `getHistoryData`
3. Должна добавлять: `metric_${metricId}: value`

**Пример правильного ответа:**
```json
{
  "success": true,
  "data": [
    {
      "timestamp": "2026-03-17T12:30:00+03:00",
      "metric_1": 22.5,
      "metric_2": 23.1,
      "metric_3": 45.2
    }
  ]
}
```

## 💡 ТИПИЧНЫЕ ПРОБЛЕМЫ:

### 🔴 Проблема 1: Нет исторических данных (но текущие есть!)

**Логи:**
```
✅ Данных получено: 0 записей
```

**Причина:** В InfluxDB есть ТОЛЬКО последнее значение, но нет истории

**Решение:** Подождите несколько минут или добавьте тестовые данные за прошлый период

---

### 🔴 Проблема 2: Backend не преобразует данные

**Логи:**
```
✅ Данных получено: 50 записей
🔍 Первая запись: {timestamp: "..."}
🔑 Найдены ключи метрик: []
❌ КРИТИЧНО: В данных НЕТ полей metric_X!
```

**Причина:** Backend получает данные из InfluxDB, но не добавляет `metric_X`

**Решение:** Проверьте код backend/src/routes/data.ts
